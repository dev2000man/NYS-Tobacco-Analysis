### Data files used in Case Study
1. The source of this dataset is a New York State government health site - in this case,
   'Active Retail Tobacco and Vapor Product Vendors'
   
    https://health.data.ny.gov/Health/Active-Retail-Tobacco-and-Vapor-Product-Vendors/9ma3-vsuk/about_data

   Of course, this is before the recent surge of CBD type vendors, which would change the study considerably if redone now.
